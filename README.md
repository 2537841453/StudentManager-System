## android版学生管理信息系统
### 主要功能如下
#### 管理员：
   -  登录，注册管理员
   - 查询学生信息
   - 添加学生信息
   - 查看成绩排名
   - 修改学生信息
   - 删除学生信息
   - 强制下线

#### 学生：
   - 登录
   - 查看个人详细信息
   - 修改密码
   - 使用androidStudio自带的sqlite数据库进行实现的

### UI部分
     没做太多的修饰，主要用了圆形头像，自定义圆角button按钮，
     自定义Alertdialog布局,然后就是相对布局和线性布局
